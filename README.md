## 출처 표기 (공공누리 §1)

본 시연 사이트는 교육부·KERIS(한국교육학술정보원)·KEDI(한국교육개발원)·KRIVET(한국직업능력연구원)에서 공공누리 제1유형으로 개방한 자료를 활용하였으며, 각 출처 URL에서 무료로 다운로드 가능합니다.

## 시연 안내

본 사이트의 17 시도 분포·검증셋 진행률·정책 효과 시뮬레이션은 **개념 시각화 및 시연 시뮬레이션**이며, 실측 데이터는 Phase 0 PoC (시범 1교 M1~M3) 완료 시점에 학교알리미 OpenAPI 및 자체 검증셋 측정으로 확정 예정입니다.
